# ST344
Code for ST344 Individual Assignment Report

**ST344ReportCode.rmd** contains all code used in individual assignment report

**ST344Geocoding.rmd** contains code for reverse geocoding, getting London districts (Boroughs) from non-unique anonymised co-ordinates


Data for report code can be downloaded in the in stopSearch_assignment.rda

ONS Demographics Data found at: https://discover.ukdataservice.ac.uk/series/?sn=200002
